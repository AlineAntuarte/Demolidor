# Demolidor
Demolidor: um remake do clássico Breakout do Atari 2600. O jogador controla uma barra horizontal para rebater a bola e destruir tijolos. Um desafio de resistência com ondas infinitas de blocos, velocidade progressiva e recordes. Desenvolvido com HTML, CSS e JS puros para a disciplina de Programação Frontend I do IFSC.
